Hal yang dikumpul
- jawaban (inget harus pake view, controler, handler,, repository, factory, dan model)
	-View (tampilan)
	-Controller (validasi input dan manggil method untuk proses backend)
	-Handler (bisnis logic, mendelegasikan perintah query(select,insert,dll) ke repository
	-Repository (berhubungan dengan model dan data daam database)
	-Factory (buat bikin object yang kompleks seperti ketika ingin create object yang memiliki hubungan dengan object lain)
	-Model (model data yang bisa digunakan untuk memanipuasi data dalam database)
